Hello! Welcome!!

I am

siyan.li@columbia.edu

Computer Science Ph.D. Candidate at Columbia University.

My absolutely fantastic advisors are Zhou Yu and Julia Hirschberg!! I am interested in Dialogue Systems applications in Education, Privacy, and Healthcare. I want to develop virtual agents that communicate with humans better.

I did my BS at Georgia Tech and started doing NLP research with the amazing Mark Riedl. I then completed my MSCS at Stanford, where I was fortunate enough to collaborate with Chris Manning and Chris Potts. I have been a teaching assistant for both Stanford's CS 224N and CS 224U, and a member of Stanford's Alexa Prize Socialbot Challenge team of 2022-2023.

My Chinese Name's Pinyin: lǐ sī yàn ( "Siyan" pronunciation).

You can find me on [ Twitter], [ LinkedIn (Very not updated)], and [ Google Scholar]. You can also take a gander at my CV.

News

  • PAPILLON
  • The code release for PAPILLON is finally here! PAPILLON allows a laptop-hosted model like Llama-3.2-3B to use more powerful API models like GPT-4o-mini as a tool, seeking to avoid (not yet 100% prevent) leaking private PII. Still very early/experimental! See the GitHub repository here.
  • EDEN
  • We release EDEN, an empathetic and encouraging spoken English conversation partner! We additionally release our trained models, as well as a small user interface to be run locally so that you can try out EDEN! See the code for EDEN here.
  • arxivParser
  • I made arxivParser! It is for parsing daily arxiv emails into audio clips by specifying keywords, finding relevant articles, then generating text-to-speech audio from the title, author, and abstract. Find it here.
